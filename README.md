## Setup

### Clone the Git repository

    git clone https://github.com/AliceO2Group/alice-fit-fpga.git

### Update to latest version

    git pull --recurse-submodules

----
### Set up Vivado 2018.1

    source /opt/Xilinx/Vivado/2018.1/settings64.sh
